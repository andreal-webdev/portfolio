About Me

My name is Andrea Larsen. I'm a web designer and developer who after working in the food industry for approximately 7 years, decided to change career. Despite my love for cooking and cake decorating, I realized that I wanted to find a new passion, something that alows me to continue using my creativity, but in a different way. After doing lots of research to explore other options, I made the decision to enroll in the Multimedia Design and Development Program at Humber College. I'm currently in the last semester of this amazing program where I discovered a passion for coding. My goal is to work as a front-end developer once I graduate. And while I acknowledge that I have much more to learn, I eagerly look forward to the journey.

Spacing Out

This is my most recent project where I explored NASA's API, bootstrap, 3D models and AR. I decided to create this astronomy website to expand my knowledge of how to use APIs, Bootstrap, Node.js and model viewer. If you want to learn more about this project you can check my Medium Article.

Pacman Maze

I recreated the original "Pacman Game" but as a maze using HTML canvas and JavaScript. This game different from the original game since there are no ghosts to chase you. Instead, it focuses on the main goal of collecting dots within the maze. The goal of this project was learn about canvas and the DOM. I want to update it and improve it by making it responsive, and giving it a score.

Ballet Studio Website

For this project I used HTML, CSS, and JavaScript. The goal of this project was to create a responsive website using bootstrap. This website was designed to be visually appealing, easy to navigate, and responsive, ensuring an optimal viewing experience on various devices.

Weatherly

This is a weather app where the data was sourced from the OpenWeather API. The goal of this project was learning about how to fetch data from an API. In this project I also used HTML, Vue.js and Tailwind. I created this projects to learn these frameworks. In this app you'll be able to check the current weather, and a 5 day forecast.
